# Electron Forge + React + Tailwind CSS Quick Start

Template khởi đầu cho ứng dụng Electron với React và Tailwind CSS, đã được cấu hình sẵn.

## 🚀 Quick Start

### Yêu cầu
- Node.js (>= 16.x)
- npm hoặc yarn

### Cài đặt & Chạy

1. Clone repository:
```bash
git clone https://github.com/pillrock/ElectronForgeViteTypeScript-TailwindCss-React--QuickStart .
```

2. Cài đặt dependencies:
```bash
npm install
```

3. Chạy ứng dụng:
```bash
npm start
```

## 🛠 Công nghệ sử dụng

- [Electron Forge](https://www.electronforge.io/) - Framework để build ứng dụng desktop
- [React](https://reactjs.org/) - UI library
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript
- [Vite](https://vitejs.dev/) - Build tool

## 📝 Lưu ý

- Sử dụng các class của Tailwind trong components React
- Khởi động lại server sau khi thay đổi cấu hình

## 🔧 Troubleshooting

Nếu gặp lỗi:
1. Xóa thư mục `node_modules` và file `package-lock.json`
2. Chạy `npm install` để cài đặt lại dependencies
3. Khởi động lại development server
